# leDouSystem

项目背景
仿中国移动游戏平台，提供了很多游戏，给用户下载，下载后需要扣费，扣话费

仿照了移动充话费，送M值----》兑换很多东西

用户扣费后，赠送乐豆，乐豆可以兑换游戏(一方面是用于回馈用户，另外一
方面促进消费)

下载游戏，可以使用话费，也可以使用乐豆，一款游戏定价(话费，乐豆)

如果是下载的时候使用乐豆兑换，则直接扣乐豆，如果使用话费下载，则既要扣话费
也要赠送乐豆
